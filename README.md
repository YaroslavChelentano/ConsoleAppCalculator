# ConsoleAppCalculator
This repository contains console application that will have functionality for sum, div, mul and sub numbers. Additionaly check for even, percentage calculator, sqrt of number, power of number. Project will work on .NET10, it will have menu with options. Have fun;)

## Git flow
- There will be three environments: development, stage, production
- development - development, stage - stage, production - main
- Each task like new features should be named in branch like features/NUMBEROFTASK
- Each bug should be named in branch like bugs/NUMBEROFTASK
- Each hotfix will be named in branch like hotfix/NUMBEROFTASK
